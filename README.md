
## API Reference
Убедись, что установлен Nodejs

```bash
  server port =http://localhost:5000
  client port =http://localhost:5173

```
## Installation

Запуск клиента
```bash
  cd client
```
Установка зависимостей
```bash
  npm install
```
Запуск
```bash
  npm run dev
```
Запуск сервера
```bash
  cd backend
```
Установка зависимостей
```bash
  npm install
```
Запуск
```bash
  npm run start
```

## Технологии

React(Vite), Nodejs, TailwindCSS, Headless UI
